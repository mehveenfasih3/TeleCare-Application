## User Requirements and Features

### General Users (No Internet Required)

* Users who do not have internet access can use the **Offline Support** module.
* This module provides general telecare information.
* Users can also use the **Lung Disease Prediction System** by entering age, gender, and other symptoms to receive real-time predictions.

### Emergency Users

* Emergency users can fill out a **quick emergency form** without signing in.
* This form is used for **emergency room booking**.

### Nurse Module

* Nurses can create, edit, and manage rooms and beds.
* Nurses can assign beds and rooms to emergency patients.
* Nurses can view and manage medicines.
* Nurses can add medicines to the system.
* Nurses can view a **daily health tip** for awareness and education.

### Patient Module

* Patients can book **on-site or off-site appointments** using an AI assistant.
* The AI assistant speaks automatically, asks questions, and fills in the appointment form.
* Patients can view all their appointments.
* For off-site appointments, patients can join a video consultation with the doctor.
* Patients can use the **Lung Disease Prediction Model** and view the confidence score (low or high risk).
* Patients can log out, and their session is destroyed securely.

### Doctor Module

* Doctors can view available appointments.
* Doctors can join video meetings through the application.
* Doctors can log out securely.
